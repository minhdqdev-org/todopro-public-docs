# Integrations

Integrations connect TodoPro to the other tools where your work lives.
Instead of checking several apps, you bring that work into TodoPro — and, where the integration supports it, push your TodoPro tasks back out.

You manage every connection in one place: **Settings → Integrations**.
From there you connect an account, choose what to sync, check the connection status, and disconnect when you no longer need it.

## The Hub

!!! info "Pro feature"
    This requires a TodoPro Pro subscription. See [TodoPro Pro](../pro/index.md).

The **Hub** is a single unified feed of the external items from all of your connected integrations.
Open it from the primary navigation in the web app.

Each item in the Hub shows where it came from, its current state, and a link back to the original item in the source tool.
The Hub is deliberately provider-agnostic: as you connect more integrations, they all land in the same feed.

## Available integrations

| Integration | What it does | Plan |
|---|---|---|
| [GitHub Projects](github-projects.md) | Browse your GitHub Projects, link one to a TodoPro project, import items, push tasks to GitHub, and keep both sides in sync. | Pro |
| [Google Calendar](calendar.md) | Push tasks to a calendar, pull events into TodoPro, and keep them in sync. | Free |
| Capy | Capy is a personal relationship manager. Connect it to bring your follow-ups into TodoPro; completing the task reports the result back to Capy. | Pro |

TodoPro also offers a read-only **calendar feed** for each project that you can subscribe to in any calendar app.
See [Calendar](calendar.md).

!!! note
    GitHub Projects, Capy, and the Hub require Pro (an active 7-day trial counts).
    Google Calendar and project calendar feeds are available on every plan.

## Connecting and disconnecting

1. Go to **Settings → Integrations**.
2. Find the integration you want and follow the connect steps for it.
3. Once connected, the section shows the connection status and the actions available for that integration.
4. To remove a connection, use **Disconnect** in the same section.

Disconnecting stops all syncing for that integration.
Anything already imported into TodoPro stays where it is.
